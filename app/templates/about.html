I WanTo Know

"I WanTo Know" is a small project where, using Wolfram|Alpha API, you can get to know some fun things.
This project was developed with the purpose of investigating technologies like:

  • Django - Framework used
  • XML - Wolfram|Alpha API
  • XML Schema - Used to validate XML from API
  • XPath - Used to get wanted result/XSLT and XQuery
  • XSLT - The result is build with XSLT
  • BaseX - XML database
  • XQuery - To manage database
  • RSS/Atom - Reddit news come from Atom feed
