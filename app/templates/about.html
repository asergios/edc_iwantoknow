I WanTo Know

"I WanTo Know" is a small project where, using Wolfram|Alpha API, you can get to know some fun things.
This project was developed with the purpose of investigating technologies like:

  • Django - Framework used
  • XSLT - Used to transform XML data to RDF/XML
  • RDF - This database is organized in RDF/XML
  • Triplestore GraphDB - Database used
  • SPARQL - Queries used to manage database and Wikidata
  • RDFa - Results are shown with RDFa semantic logic
  • Wikidata - The images you see of people and locations are gotten from Wikidata
